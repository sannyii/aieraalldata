# 使用指南

## 快速开始

1. **安装依赖**
   ```bash
   npm install
   ```

2. **准备数据文件**
   - 在 `data` 文件夹下的对应月份文件夹中放置 Excel 文件
   - 例如：`data/202511/` 文件夹中放置 2025年11月的数据文件

3. **启动应用**
   ```bash
   npm run dev
   ```

4. **访问应用**
   - 打开浏览器访问 http://localhost:3000
   - 在右上角选择要查看的月份
   - 系统会自动显示该月数据以及与上个月的对比

## 数据文件要求

### 文件夹结构
```
data/
├── 202501/     # 2025年1月
│   └── 数据.xlsx
├── 202502/     # 2025年2月
│   └── 数据.xlsx
...
└── 202512/     # 2025年12月
    └── 数据.xlsx
```

### Excel 文件格式

Excel 文件必须包含以下列（第一行为表头）：

| 列名 | 说明 | 示例 |
|------|------|------|
| 账号名称 | 账号名称 | 虎嗅APP |
| 发文数 | 发文数量 | 353 |
| 总阅读数 | 总阅读数（支持"w"单位） | 1176w+ |
| 头条阅读 | 头条阅读数 | 857w+ |
| 10万+ | 10万+文章数 | 48 |
| 平均阅读 | 平均阅读数 | 3.3w |
| 总在看数 | 总在看数 | 2.5w |
| 总点赞数 | 总点赞数 | 6.8w |
| 总转发数 | 总转发数 | 46w+ |
| WCI/排名 | WCI值或排名 | 77 |

### 数据格式说明

- 数字可以包含 "w" 表示"万"（如：`3.3w` = 33000）
- 数字可以包含 "+" 符号（会被自动忽略）
- 账号名称必须唯一，用于匹配不同月份的数据

## 功能说明

### 数据对比

- **增长数据**：用红色方框标识，显示 "+数值"
- **减少数据**：用绿色方框标识，显示 "-数值" 或直接显示负数
- **无变化**：不显示变化标识

### 月份选择

- 右上角下拉菜单显示所有可用的月份
- 月份按时间倒序排列（最新的在前）
- 选择月份后自动加载数据并计算与上个月的对比

### 响应式设计

- 页面占满整个屏幕，无滚动条
- 表格区域可滚动查看所有数据
- 自适应不同屏幕尺寸

## 常见问题

### Q: 为什么看不到数据？
A: 请检查：
1. `data` 文件夹中是否有对应月份的文件夹
2. 月份文件夹中是否有 Excel 文件
3. Excel 文件格式是否正确

### Q: 为什么对比数据不正确？
A: 请确保：
1. 不同月份的 Excel 文件中，账号名称完全一致
2. 数据列的顺序和名称与要求一致

### Q: 如何添加新的月份数据？
A: 只需在 `data` 文件夹下创建新的月份文件夹（如 `202601`），并放入 Excel 文件即可。
