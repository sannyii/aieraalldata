# 数据文件夹说明

## 文件夹结构

本文件夹包含 12 个月份的子文件夹：
- `202501/` - 2025年1月
- `202502/` - 2025年2月
- `202503/` - 2025年3月
- `202504/` - 2025年4月
- `202505/` - 2025年5月
- `202506/` - 2025年6月
- `202507/` - 2025年7月
- `202508/` - 2025年8月
- `202509/` - 2025年9月
- `202510/` - 2025年10月
- `202511/` - 2025年11月
- `202512/` - 2025年12月

## 使用方法

1. 在每个月份文件夹中放置一个 Excel 文件
2. Excel 文件名可以任意，但每个文件夹只能有一个 Excel 文件
3. Excel 文件应包含以下列（第一行为表头）：
   - 账号名称
   - 发文数
   - 总阅读数
   - 头条阅读
   - 10万+
   - 平均阅读
   - 总在看数
   - 总点赞数
   - 总转发数
   - WCI/排名

## 示例

```
data/
├── 202501/
│   └── 竞家数据.xlsx
├── 202502/
│   └── 竞家数据.xlsx
└── 202503/
    └── 竞家数据.xlsx
```

启动应用后，系统会自动读取这些文件夹中的 Excel 文件，并在页面上显示。
