# 竞家数据统计看板

基于 Next.js 的数据对比分析看板，用于展示每月竞家数据统计及变化趋势。

## 功能特性

- 📊 数据可视化展示
- 📈 月度数据对比（增长/减少标识）
- 📁 Excel 文件上传和解析
- 📱 响应式设计，适配各种屏幕尺寸
- 🎨 直观的颜色标识（增长红色，减少绿色）

## 技术栈

- Next.js 14
- React 18
- TypeScript
- xlsx (Excel 文件解析)

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000)

### 构建生产版本

```bash
npm run build
npm start
```

## 使用说明

### 数据文件放置

1. 在项目根目录下的 `data` 文件夹中，按照月份创建子文件夹：
   - `data/202501/` - 2025年1月
   - `data/202502/` - 2025年2月
   - `data/202503/` - 2025年3月
   - ... 依此类推到 `data/202512/` - 2025年12月

2. 在每个月份文件夹中放置一个 Excel 文件（文件名不限，但每个文件夹只能有一个 Excel 文件）

3. 启动应用后，在右上角选择要查看的月份

4. 表格会自动显示选中月份的数据，以及与上个月的对比变化

5. 增长的数据用红色方框标识，减少的数据用绿色方框标识

## Excel 文件格式要求

Excel 文件应包含以下列（第一行为表头）：

**基础信息：**
- 公众号
- 帐号名

**文章相关：**
- 文章总数
- 文章总增量
- 超10W文章数
- 超10W文章数增量
- 发布次数
- 发布次数增量

**阅读相关：**
- 阅读总数
- 阅读总数增量
- 平均阅读数
- 平均阅读数增量
- 头条文章阅读量
- 头条文章阅读增量
- 最大阅读数

**推荐相关：**
- 推荐总数
- 推荐总数增量
- 平均推荐数
- 平均推荐数增量
- 头条文章推荐数
- 最大推荐数
- 推荐率

**点赞相关：**
- 点赞总数
- 点赞数增量
- 最大点赞数
- 头条文章点赞总数
- 平均点赞数
- 平均点赞数增量

**转发相关：**
- 转发总量
- 最大转发数
- 头条转发总数

**排名相关：**
- WCI
- WCI增量
- 总排名
- 总排名变化

## 文件夹结构示例

```
data/
├── 202501/
│   └── 数据.xlsx
├── 202502/
│   └── 数据.xlsx
├── 202503/
│   └── 数据.xlsx
...
└── 202512/
    └── 数据.xlsx
```

每个月份文件夹中只需要放置一个 Excel 文件即可，文件名可以任意。
